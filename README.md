# German statistic visualization

## Typical metrics for comparing Germany’s states include:

### Demographic:
- Population
- Population density
- Age distribution

### Economic
- GDP / GDP per capita
- Unemployment rate
- Average income

### Social
- Education levels
- Crime rates
- Number of universities

### Infrastructure
- Internet speed / broadband coverage
- Healthcare facilities per capita

### Environmental
- CO₂ emissions
- Share of renewable energy
